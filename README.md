Hilera de caracteres: es una secuencia ordenada de elementos tales como letras, numeros, simbolos y espacios. Usados para representar texto.
Ejemplos de hilera de caracteres: Contar cantidad de caracteres en la hilera, buscar la primera o ultima letra de alguna cadena.
Diferencia entre un caracter y una hilera: Un caracter es una unidad, es decir, una sola letra, numero o simbolo. Una hilera es una secuencia con varias letras, numeros o simbolos.
Operaciones que se pueden realizar con una hilera: 
1. Pasar a mayúscula o minúscula
2. Editar
3. Buscar la aparición de una letra o sub-hilera dentro de otra hilera.
4. Obtener la letra que se encuentra en una posición dada.
5. Conocer el tamaño de una hilera.
6. Extraer un fragmento de texto desde una hilera.
7. Comparar el contenido de dos hileras de caracteres.
8. Comparar el contenido de dos hileras de caracteres ignorando entre mayúsculas y minúsculas
